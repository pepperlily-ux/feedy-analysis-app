# 🚀 Feedy 실행하기 (초보자 가이드)

## Step 1: Node.js 설치 확인

터미널을 열고 아래 명령어를 입력하세요:

```bash
node --version
```

버전이 나오면 OK! (예: v18.0.0)
안 나오면 https://nodejs.org 에서 설치하세요.

---

## Step 2: 프로젝트 폴더로 이동

```bash
cd feedy
```

---

## Step 3: 패키지 설치

```bash
npm install
```

처음 한 번만 하면 됩니다! (1-2분 소요)

---

## Step 4: 개발 서버 실행

```bash
npm run dev
```

---

## Step 5: 브라우저에서 확인

터미널에 나오는 주소로 접속하세요!
보통 `http://localhost:3000` 입니다.

---

## 🎉 완료!

이제 Feedy를 사용할 수 있습니다!

### 사용 방법:
1. **Home** 페이지에서 피드백 입력
2. **Cards** 페이지에서 피드백 목록 확인
3. **List** 페이지에서 AI 추천 기능 확인

---

## ❓ 문제 해결

### 포트가 이미 사용 중이라고 나와요
다른 프로그램이 3000 포트를 쓰고 있습니다.
`vite.config.js`에서 포트를 바꿔보세요 (예: 3001)

### 패키지 설치가 안돼요
인터넷 연결을 확인하고 다시 시도해보세요.

### 화면이 안 나와요
브라우저 콘솔(F12)을 열어서 에러를 확인해보세요.

---

## 🛠️ 개발 팁

- 코드를 수정하면 자동으로 새로고침됩니다
- `Ctrl + C`로 서버를 종료할 수 있습니다
- `src/` 폴더 안의 파일을 수정하면 됩니다
